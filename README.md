# Monopoly-Game
a non graphical Java version of the classic Monopoly game.

# How To Run
    Unix
    -- javac -cp .:jsonsimple.jar *.java
    -- java -cp .:jsonsimple.jar Main "commandFilePath"
    
    Windows
    -- javac -cp .;jsonsimple.jar *.java
    -- java -cp .;jsonsimple.jar Main "commandFilePath"
